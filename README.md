# telegram-mirror-bot

## Setup
1. make `.env` file
  ```ini
  TOKEN = "<Telegram Bot Token>"
  ```
2. make `praw.ini` file
  ```ini
  [telegram mirror bot]
  client_id=<Reddit Client ID>
  client_secret=<Reddit Client Secret>
  username=<Reddit Account Username>
  password=<Reddit Account Password>
  user_agent=tg mirror bot 0.1
  ```
